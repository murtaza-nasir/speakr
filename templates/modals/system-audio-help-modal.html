Browser Compatibility

System audio recording requires a Chromium-based browser (Chrome, Edge, Brave) and is not supported on mobile devices.

How to record system audio:

  1. Click "System Audio" or "Mic + System" button
  2. A screen sharing dialog will appear
  3. Select a Chrome tab or entire screen
  4. Make sure "Share audio" checkbox is enabled
  5. Click "Share" to start recording

Troubleshooting:

  • Ensure you're using Chrome, Edge, or Brave
  • Check that "Share audio" option is visible and enabled
  • Some content may have DRM protection that blocks recording
  • Firefox and Safari do not support this feature
